## Cara Menjalankan Program C:

Pastikan kamu menginstal SDL2 dan SDL2_ttf.

Kompilasi program ini menggunakan perintah berikut:

gcc -o quote_maker quote_maker.c -lSDL2 -lSDL2_ttf


Jalankan program:

./quote_maker


## Cara Menjalankan Program Haskell:

Pastikan kamu sudah menginstal Gloss.

Kompilasi program ini dengan perintah berikut:

ghc -o quote_maker quote_maker.hs


Jalankan program:

./quote_maker
